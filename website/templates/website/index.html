Projet finance • Machine Learning

Crédit scoring

Ce projet porte sur le développement d’une application web de credit scoring permettant d’évaluer automatiquement le risque de défaut d’un emprunteur grâce à l’intégration d’un modèle de machine learning dans une architecture Django.

L’application exploite les données financières et personnelles des demandeurs de crédit — telles que le revenu, le montant du prêt, le taux d’endettement ou encore l’historique de crédit — afin d’estimer une probabilité de défaut et de générer une décision d’aide à l’octroi : accepté, à examiner ou refusé.

Le projet s’inscrit dans une démarche complète de data science, allant du nettoyage des données et de l’analyse exploratoire jusqu’au feature engineering, à l’entraînement et à l’évaluation de plusieurs modèles de classification, notamment la régression logistique, Random Forest et Gradient Boosting.

La sélection du modèle repose sur des indicateurs de performance pertinents comme le ROC-AUC, la précision, le rappel et le F1-score, afin de garantir une évaluation robuste et adaptée aux enjeux métier liés au risque de crédit.

Un dashboard analytique complète l’application en offrant une visualisation claire des indicateurs clés du portefeuille, comme la probabilité moyenne de défaut, le taux d’acceptation et la distribution des scores de risque. Les résultats sont stockés dans PostgreSQL et peuvent être exportés en CSV ou en PDF.

Technologies utilisées : Python, Django, PostgreSQL, Pandas, NumPy, Scikit-learn, XGBoost, Matplotlib et Seaborn.

Ce projet consiste en une application de détection automatique de mails malveillants, conçue pour identifier les messages suspects, frauduleux ou indésirables à partir de leur contenu textuel.

Projet IA • Django • RAG

AssurBot

AssurBot est une application web intelligente développée avec Django, intégrant une architecture RAG (Retrieval-Augmented Generation) pour fournir des réponses précises et contextualisées dans le domaine de l’assurance (sinistres, garanties, démarches, etc).

Le projet combine un backend robuste en Python avec une interface moderne, offrant une expérience utilisateur fluide et interactive.

Côté intelligence artificielle, le système repose sur l’utilisation de SentenceTransformers pour générer des embeddings, associés à FAISS pour effectuer une recherche vectorielle rapide et pertinente dans une base documentaire.

Les informations récupérées sont ensuite exploitées par un modèle de génération via l’API OpenAI afin de produire des réponses naturelles et contextualisées.

Le backend Django gère les requêtes utilisateurs, l’orchestration du pipeline RAG, ainsi que la gestion des conversations stockées en base de données.

Le frontend utilise HTML, CSS et JavaScript pour proposer une interface dynamique avec envoi asynchrone, affichage progressif des réponses et animations type chatbot moderne.

Le projet met en œuvre une architecture modulaire incluant un retriever et un generator (LLM), permettant de séparer clairement la recherche d’information et la génération de réponses.

Technologies utilisées : Python, Django, OpenAI API, SentenceTransformers, FAISS, HTML, CSS, JavaScript, SQLite.

Ce projet démontre la capacité à concevoir une application full-stack intégrant de l’intelligence artificielle avancée, avec une attention particulière portée à la performance, à la pertinence des réponses et à l’expérience utilisateur.
